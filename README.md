# 🌤️ Weather Forecast App

A modern weather forecasting application built with **React.js** that provides real-time weather information and a **7-day weather prediction** for any location around the world.

## 📖 Overview

Weather Forecast App allows users to search for cities worldwide and view detailed weather information, including current conditions and future forecasts. The application offers a clean, responsive, and user-friendly interface for tracking weather updates anytime and anywhere.

## ✨ Features

* 🌍 Search weather for any city in the world
* 🌡️ Display current temperature and weather conditions
* 📅 7-day weather forecast prediction
* 🌅 Sunrise and sunset information
* 💨 Wind speed and direction
* 💧 Humidity level
* 👀 Visibility information
* 🌦️ Weather icons based on current conditions
* 📱 Fully responsive design for mobile, tablet, and desktop
* ⚡ Fast and interactive user experience with React.js

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* Weather API
* Axios / Fetch API
* HTML5
* CSS3

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/meankunthea/weather_app_reactjs.git
```

### Navigate to the project directory

```bash
cd weather_app_reactjs
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will run at:

```bash
http://localhost:5173/
```

## 🔍 How to Use

1. Open the application.
2. Enter a city name in the search bar.
3. Press **Search** or hit **Enter**.
4. View the current weather information.
5. Check the 7-day forecast prediction.
6. Search another city to get weather updates instantly.
